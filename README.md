# CAS
A CAS (Computer Algebra System), whichh is able to perform operations on algebraic expressions, like:
* simplifying them (the only actually implemented thing currently)
* solving (in)equalities (WIP)
* calculating derivatives and integrals
* Graphing
And so much more.

This is not something commertial or that I am making for use by other people, rather just a students project. If you want an actual good CAS, use WolframAlpha or SymPy if you need a library for python
